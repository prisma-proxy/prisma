# 新手指南

欢迎来到 Prisma 新手指南！如果你从未使用过代理 (Proxy)、从未配置过服务器，甚至听到"加密 (Encryption)"这个词就感到紧张——这份指南就是为你准备的。

我们将**从零开始**，一步一步地讲解所有内容，配以大量的解释、类比和图表。当你完成本指南后，你将拥有一个完全可用的 Prisma 设置，它能让你的网络流量保持私密和安全。

## 什么是 Prisma？

Prisma 是一个在你的电脑和互联网上某台服务器之间创建**加密隧道 (Encrypted Tunnel)**的工具。你所有的网络流量都通过这条隧道 (Tunnel) 传输，所以中间的任何人——不管是你的网络服务提供商 (ISP)、你的学校或公司网络，还是任何其他人——都无法看到你在网上做什么。

你可以这样理解：

> 想象你要给朋友寄一封信，但你不想让邮递员看到信的内容。所以你把信放进一个**上了锁的箱子**里，只有你的朋友有密钥 (Key)。这本质上就是 Prisma 对你的网络流量所做的事情。

## 为什么你需要 Prisma？

人们使用 Prisma 这样的工具有很多原因：

- **隐私** —— 防止网络服务提供商查看你的浏览活动
- **安全** —— 在公共 Wi-Fi（咖啡厅、机场、酒店）上保护你的数据
- **访问** —— 访问可能在你的网络上被屏蔽的网站和服务
- **自由** —— 绕过互联网审查和过滤

## 前提条件

你只需要两样东西：

1. **一台电脑** —— Windows、macOS 或 Linux
2. **网络连接**

就这些。其他所有内容我们都会在过程中解释。

本指南假设你对网络、服务器或命令行**完全没有**任何基础知识。每个概念都从头开始解释。如果有什么地方不清楚，那是我们的问题，不是你的。

## 如何使用本指南

- **按顺序阅读** —— 每个章节都建立在前一章的基础上
- **不要跳过基础** —— 即使你想跳过，[基础概念章节](./basics.md)也会帮助你理解后面的所有内容
- **动手实践** —— 最好的学习方式是实际操作。跟着示例一步步做
- **不用担心犯错** —— 本指南中的任何操作都不会损坏你的电脑。如果出了问题，你随时可以重新开始
